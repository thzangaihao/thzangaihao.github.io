01. Pymol 常用指令汇总

表面静电势能计算着色

计算真空静电势,mode=2生成protein_obj_e_map对象，注意真实情况不能用真空，需要使用 APBS (Adaptive Poisson-Boltzmann Solver) 进行静电势精确计算。


util.protein_vacuum_esp("protein_obj", mode=2, quiet=0)

标签显示方案

氨基酸显示方案

显示“单字母 + 编号”（例如 A50）选择需要标记的氨基酸：


# 格式为：单字母 + 编号 (如 W120)，显示在α碳原子上 (CA)
label target_res and n. CA, "%s%s" % (oneletter, resi)

标签位置调整

注意：[X, Y, Z] 是相对于该原子坐标的偏移量。


# 以 CA 为坐标原点，给 50 号残基设置一个偏移
set label_position, [2, 2, 0], resi 50 and n. CA

标签美化

对所有标签统一设置：


美化：调整字体大小、颜色和位置
set label_size, 20
set label_color, yellow
set label_position, [1, 1, 1]

使用 Editing Mode 按住左 Ctrl 用鼠标右键拖动以此来精细化调节标签。

标签删除

对于单个标签的精确删除：


# 删除 50 号残基 CA 原子上的标签
label resi 50 and n. CA, ""

对于所有标签的删除：


# 清除所有标签
label all, ""

蛋白对齐

我们可以使用 align obj_1, obj_2 进行对齐：


align obj_1, obj_2

材质设置

pymol 自带的材质会反光，给人一种油亮亮的感觉，我不太喜欢，我们可以用这几个参数：


# 控制反射光的强度。设为 0 即完全不反射光源。
set spec_power, 0
# 控制高光的集中程度。
set spec_reflect, 0

颜色设置

我们首先要把自定义颜色存入变量中：


set_color color_name, [74, 92, 120]

再将其设置给对象：


color color_name, object_name

光照设置

我们可以更改环境光亮度：


set ambient, 0.5

调高 ambient（环境光）会让模型看起来更“扁平”，类似手绘漫画风格，但是会使其减弱立体效果。

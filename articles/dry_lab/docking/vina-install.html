AutoDock Vina 安装与环境配置

2026-01-10 分子对接

本文记录了在 Ubuntu 22.04 下编译安装 AutoDock Vina 的全过程。

1. 下载源码

首先从 GitHub 克隆仓库：

git clone https://github.com/ccsb-scripps/AutoDock-Vina.git

2. 编译环境

确保安装了 Boost 库：

sudo apt-get install libboost-all-dev swig

3. 数学原理回顾

在安装前，我们先复习一下对接打分函数中的高斯项公式：

$$ gauss(d) = w_{gauss} \cdot e^{-(d/ \sigma)^2} $$

4. 常见报错

如果在编译时遇到 fatal error: boost/version.hpp，请检查 include 路径。

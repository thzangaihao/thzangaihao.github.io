01. Pymol常用指令汇总

1. 表面静电势能计算着色

计算真空静电势,mode=2生成protein_obj_e_map对象，注意真实情况不能用真空，需要使用 APBS (Adaptive Poisson-Boltzmann Solver) 进行静电势精确计算。


util.protein_vacuum_esp("protein_obj", mode=2, quiet=0)

2. 标签显示方案

2.1 氨基酸显示方案

显示“单字母 + 编号”（例如 A50）选择需要标记的氨基酸：


# 格式为：单字母 + 编号 (如 W120)，显示在α碳原子上 (CA)
label target_res and n. CA, "%s%s" % (oneletter, resi)

2.2 标签位置调整

注意：[X, Y, Z] 是相对于该原子坐标的偏移量。


# 以 CA 为坐标原点，给 50 号残基设置一个偏移
set label_position, [2, 2, 0], resi 50 and n. CA

2.3 标签美化

对所有标签统一设置：


美化：调整字体大小、颜色和位置
set label_size, 20
set label_color, yellow
set label_position, [1, 1, 1]

使用 Editing Mode 按住左 Ctrl 用鼠标右键拖动以此来精细化调节标签。

2.4 标签删除

对于单个标签的精确删除：


# 删除 50 号残基 CA 原子上的标签
label resi 50 and n. CA, ""

对于所有标签的删除：


# 清除所有标签
label all, ""

05. IQ TREE

简介

IQ-TREE 就是用来建树的，你可以理解它为 MEGA 软件的建树部分。虽然 MEGA 可以跑 NJ、ML、MP 等多种算法，但 IQ-TREE 专注于最大似然法 (Maximum Likelihood, ML)。它之所以在学术界（尤其是发高分文章时）地位极高，是因为它把 ML 建树做到了极致：

  • 自动选样：它能自动测试成百上千种进化模型，选出最符合你那堆序列的一款。
  • 超快增强值：传统 MEGA 跑 1000 次 Bootstrap 可能要跑几天，IQ-TREE 的专利算法（Ultrafast Bootstrap）几分钟就能跑完，而且更准。
  • 深度搜索：它在寻找最优树时比 MEGA 更彻底，不容易陷入局部最优解。

环境搭建及软件安装


# 1. 创建一个名为 phylogenomics 的新环境（专门用于高级建树）
conda create -n phylogenomics -c bioconda iqtree

# 2. 激活环境
conda activate phylogenomics

# 3. 验证安装（如果输出了版本号和帮助信息，就说明成功了）
iqtree -h

用法

IQ-TREE 一般用于 OrthoFinder 使用 MSA 跑出来的同源单拷贝基因蛋白对齐后的 .fasta 文件。IQ-TREE 当然也支持基因组序列，在输入之前，一定要对齐！

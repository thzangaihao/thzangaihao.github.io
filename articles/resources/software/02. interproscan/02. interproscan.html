02. interproscan

1. 简介

InterProScan (IPRSCAN) 是一款根据蛋白结构域去预测蛋白功能的软件，我们只需要输入这个蛋白的氨基酸序列，软件就可以在多个维度对蛋白进行功能性预测，输出蛋白的功能性描述，GO 数据库中的 ID，也可以输出 Reactome 数据库中的 ID。同样，如果我们获得了一些未知功能的基因序列，我们可以对其使用 gffread 将其翻译为蛋白序列，再使用 IPRSCAN 对这个基因的功能进行预测。

2. 环境搭建及软件安装

IPRSCAN 的运行需要软件和数据库两部分构成，我们先来解决软件部分的安装：

2.1 软件主体安装


# 1. 创建名为 interproscan_env 的环境，并指定 python 版本（建议 3.9+）
conda create -n interproscan_env python=3.10 -y

# 2. 激活环境
conda activate interproscan_env

# 3. 安装 IPRSCAN
conda install interproscan -y

# 4. 验证安装
interproscan.sh --version
你也可以进行外部安装，其主程序为interproscan.sh，你可以使用./interproscan.sh --version来验证安装。

2.2 数据库主体

IPRSCAN 分析需要依赖数据库，我们需要下载对应数据库并解压，数据库包含一套文件，将下载的数据库 data 文件夹放置在 InterProScan 的安装目录下，或者在 interproscan.properties 配置文件中指定 data.directory 的绝对路径。

3. 预测

使用以下交互式脚本进行分析：
fastp_batch_clean
文件格式：.py | 大小：4.4 KB

使用前记得在脚本里配置路径等。

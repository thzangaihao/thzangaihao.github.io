03. gffread

1. 简介

gffread 用于对基因组注释文件的操作，例如格式转换，序列提取等等，在这篇博客中记录了几种常见使用方法和便携脚本。

2. 环境搭建及软件安装


# 1. 创建名为 gffread_env 的环境，并指定 python 版本（建议 3.9+）
conda create -n gffread_env python=3.10 -y

# 2. 激活环境
conda activate gffread_env

# 3. 安装 IPRSCAN
conda install gffread -y

# 4. 验证安装
gffread --version

3. 常见用法

3.1 根据基因提取蛋白质序列

我们需要准备基因组.fasta文件，基因组.gtf注释文件（或其它相关注释文件例如.gff、.gff3），以及你要提取的基因名称列表的.tsv文件。gffread 会根据这些文件提取你指定的基因的蛋白质序列，生成.faa蛋白序列文件。

gffread
文件格式：.py | 大小：5.3 KB
脚本中使用参数-y输出蛋白序列。

# -y 提取蛋白序列
cmd = f"gffread '{gff_file}' -g '{ref_fasta}' -y '{temp_all_faa}'"
subprocess.run(cmd, shell=True, check=True)

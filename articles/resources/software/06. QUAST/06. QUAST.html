06. QUAST

简介

QUAST (Quality Assessment Tool for Genomes) 是一个用于评估基因组组装质量的工具。它能够对组装结果进行多方面的评估。

环境搭建及软件安装


# 1. 创建一个名为 quast 的新环境（专门用于基因组组装质量评估）
conda create -n quast -c bioconda quast

# 2. 激活环境
conda activate quast

# 3. 验证安装
quast.py --version

用法

丢进去组装好的 .fasta 文件，直接出结果。我们使用以下脚本：

quast.py

基因组爬取

提供组装 ID，下载基因组信息，包括全基因组序列文件、CDS 文件、蛋白质文件和基因组注释文件。

因为大批量基因组数据下载是非常吃网络和时间的，因此我们使用任务拆分法，同时建议申请 NCBI 的下载 API，可以将每秒请求 4 个提升为 10 个。拆封参数和 API 配置均由脚本内部完整，并自动生成 .json 配置文件。

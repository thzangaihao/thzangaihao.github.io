11. Hi-C 互作热图

脚本

根据互作分析，我们得到的核心文件是 .mcool。这是一个二进制文件，对它的解析必须要依赖环境中的软件。因此，我们采用 python matplotlib 进行绘图，这样可以避免繁重的 R 依赖。

推荐阅读：01. Hi-C 标准化流程
Hi-C 双端测序数据构建多分辨率染色质互作矩阵

这个脚本可以选择目的染色体进行绘制，也可以选择全染色体进行绘制，在不同尺度下需要设置不同的分辨率，避免热图过大卡死。出于性能考虑，热图主体部分是经过栅格化嵌入到矢量图中的。

这个脚本和其它 python 脚本有区别，参数配置逻辑在文件头部位，逻辑和 R 语言一致，不再支持交互式。
